# NestJS To-Do App
A simple todo app built using NestJS with TypeScript

## Project Setup
1. Clone repository
2. Install Node.js
3. `cd` into cloned repo and then `todo-api` directory
4. Run `npm install` to install dependencies
5. Run `npm run start` to start project
6. To test, visit `localhost:3000`
